# Tools for Building Knowledge Graphs

## Neo4j

Neo4j is a graph database that stores data as nodes and relationships. It is commonly used for building and querying knowledge graphs.

## Protégé

Protégé is an open-source ontology editor developed by Stanford University. It is used for creating ontologies and semantic models.

## GraphDB

GraphDB is a semantic graph database that supports RDF data and SPARQL queries.

## Apache Jena

Apache Jena is a Java framework used for developing Semantic Web and Linked Data applications.

## RDF

Resource Description Framework (RDF) is a standard model for representing information.

Example:

Student → studies → AI

This structure is called an RDF Triple:

* Subject: Student
* Predicate: studies
* Object: AI

## SPARQL

SPARQL is a query language used to retrieve and manipulate data stored in RDF format.
