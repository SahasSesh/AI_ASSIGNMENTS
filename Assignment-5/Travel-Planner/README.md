# AI Based Travel Planner

## Aim

To design an AI based travel planner that generates personalized travel plans using tourist places, food recommendations and cost estimation.

## Description

The planner takes the destination, number of days and budget as input. Based on the available information in the knowledge base, it suggests tourist attractions, food options and estimated travel cost.

## Working

1. Accept destination details.
2. Retrieve tourist places from the knowledge base.
3. Retrieve food recommendations.
4. Estimate travel cost.
5. Generate a travel plan.

## Result

The system generates a travel plan according to the user's requirements.
