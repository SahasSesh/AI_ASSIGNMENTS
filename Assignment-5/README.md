# Assignment 5 - Artificial Intelligence

## Objective

The objective of this assignment is to study and implement important Artificial Intelligence techniques including search algorithms, travel planning systems, knowledge graphs, and Bayesian networks.

## Tasks Completed

### Search Algorithms

The following search algorithms were implemented and tested:

- Minimax Search
- Alpha-Beta Search
- Heuristic Alpha-Beta Search
- Monte-Carlo Tree Search

Each implementation includes source code, documentation, test cases, and execution screenshots.

### AI Based Travel Planner

An AI-based travel planner was developed using a simple knowledge base containing tourist places, food recommendations, and cost estimation information. The planner generates travel suggestions based on user inputs.

### Knowledge Graphs

A study of Knowledge Graphs was carried out, including their concepts, applications, advantages, and commonly used tools such as Neo4j, Protégé, GraphDB, Apache Jena, RDF, and SPARQL.

### Bayesian Networks

A Bayesian Network was implemented to demonstrate probabilistic reasoning and inference using conditional probabilities.

## Technologies Used

- Python
- GitHub
- Visual Studio Code

## Result

All tasks specified in the assignment were successfully completed, documented, and tested.
