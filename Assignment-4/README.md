# Assignment-4: Constraint Satisfaction Problems (CSP)

This repository contains Python implementations of classic Constraint Satisfaction Problems (CSP).

---

## Problems Implemented

### 1. Australia Map Coloring

* Colors the 7 regions of Australia
* Ensures no adjacent states share the same color
* Uses backtracking

### 2. Telangana Map Coloring

* Applies map coloring to selected districts of Telangana
* Uses CSP with adjacency constraints
* Demonstrates scalability of map coloring problems

### 3. Sudoku Solver

* Solves a 9x9 Sudoku puzzle
* Uses backtracking
* Ensures row, column, and 3x3 grid constraints

### 4. Cryptarithmetic Puzzle

* Solves: SEND + MORE = MONEY
* Assigns digits to letters using permutations
* Ensures valid arithmetic solution

---

## Techniques Used

* Backtracking algorithm
* Constraint checking
* Recursive search
* Permutations (for cryptarithmetic)
